## Building your own

The pattern is four lines of HTML with a percentage. Copy this, set the widths yourself, and add `alt` or `warn` to the bar class for the purple or red variants:

```html
<div class="chart" markdown>

<div class="chart-row">
<div class="chart-label">Label</div>
<div class="chart-track"><div class="chart-bar" style="width:72%"></div></div>
<div class="chart-value">72</div>
</div>

<p class="chart-caption">What the reader should take from it, and where the number came from.</p>

</div>
```

**Rules.** Widths are proportions of the largest value in that chart, set by hand — there is no runtime calculation, so check your arithmetic. Always caption with the source. Never plot a number the underlying data page doesn't support.
